Bootstrap
Framework

Bootstrap is the world's most popular open-source CSS framework, originally created at Twitter in 2011. It provides a responsive 12-column grid system, pre-styled components (buttons, cards, modals, navbars), and utility classes — allowing developers to build mobile-first, professional-looking interfaces rapidly without writing CSS from scratch. Bootstrap 5 dropped jQuery as a dependency, relying on vanilla JS instead.

Mobile-first means: start with the small-screen layout (no suffix, or col-sm-*), then override for larger breakpoints with col-md-*, col-lg-*, etc. This is better for performance — mobile devices download the base styles and skip the larger-screen overrides.

How Bootstrap works: It ships a single CSS file (~200 KB minified) containing thousands of utility classes and component styles. Classes like d-flex, mb-3, col-md-4 are atomic — each does one thing. Combined, they eliminate the need to write repetitive custom CSS for the majority of UI patterns.

Utility Classes

Bootstrap 5's utility API provides atomic classes covering spacing, flexbox, typography, colours, borders, shadows, and display. These replace repetitive one-line CSS rules.

Spacing scale: Bootstrap uses a $spacer base of 1rem. Class numbers map as: 0 → 0, 1 → 0.25rem, 2 → 0.5rem, 3 → 1rem, 4 → 1.5rem, 5 → 3rem. So mb-4 = margin-bottom: 1.5rem.

Responsive Breakpoints

Bootstrap's grid and many utility classes use six breakpoints defined as min-width media queries. Because Bootstrap is mobile-first, each breakpoint activates at the specified width and wider.

bootstrap 5 breakpoints
PrefixMin-widthDevice target
(none)0px — all screensExtra small: portrait phones
sm≥ 576pxSmall: landscape phones
md≥ 768pxMedium: tablets
lg≥ 992pxLarge: desktops
xl≥ 1200pxExtra large: large desktops
xxl≥ 1400pxExtra extra large: wide monitors
/* Bootstrap generates classes like: */ @media (min-width: 768px) { .col-md-4 { width: 33.3333%; } .d-md-block { display: block !important; } .d-md-none { display: none !important; } }
Practical pattern: col-12 col-md-6 col-lg-4 — full width on mobile, half-width on tablet, one-third on desktop. This single class string handles three different layouts without a single media query written by hand.
